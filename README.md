## Manually
### Build
#### Front-end
```
cd $REPO/application
npm install
npm run build
```
#### Back-end
```
cd $REPO/server
npm install
```
### Run
#### Front-end
Open `$REPO/application/build/index.html` page in your browser.  

#### Back-end
```
cd $REPO/server
npm start
```
### Data Files
By default, data files are stored under "$REPO/server/data/"

## Import/Export Data Files
Simply copy the JSON data files into another project or for backup.
